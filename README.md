# ProspectSense Standalone HTML App

## Getting Started

- [ ] Install Gulp CLI
- [ ] Install Dependencies
- [ ] Run Development

Firstly you should install Gulp CLI.
```
npm install -g gulp-cli
```


Then install all dependencies,
```
npm install && npm start
```
